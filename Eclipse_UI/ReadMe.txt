Use Visual Studio code to open this project.
Download and install Visual Studio code from : https://code.visualstudio.com/Download

Open VS code. Go to File -> open folder. select Eclipse_UI folder.

-----------------------
To run application below are pre-requisites.
NodeJS (https://nodejs.org/en/)
Lite-Server (https://github.com/johnpapa/lite-server)

---------------
Run application:
open command prompt and navigate to application folder.
run >npm install
>npm start
